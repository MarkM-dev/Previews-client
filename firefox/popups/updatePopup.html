Twitch Previews Updated!

New Feature!

Live stream previews in Twitch directories (followed streams grid list, game categories, browsing, etc..)
- hover over a stream in a twitch directory (pages where there is a grid list of live streams) - the thumbnail will be temporarily replaced with the live stream preview.
* This option is turned on by default and can be changed in the extension options as shown by the red arrow in the picture below.
* If you can't see the extension icon at the top bar - click the white chrome extensions manager icon next to the Twitch Previews extension icon in the picture below.
